Compilation de FileZilla

Pour compiler FileZilla, vous avez besoin des programmes et outils suivants:

  • MS Visual C++ 6

  • La dernière version de la plate-forme MS SDK sur http://www.microsoft.com/msdownload/platformsdk/sdkupdate/

  • MS HTML Help Workshop

Vous pouvez télécharger la dernière version de HTML Help Workshop sur Microsoft Website. Après installation, il vous faudra préparer DevStudio: allez à Outils, Options, Dossiers, puis (en vous assurant d'avoir installé HTML Help Workshop sur C:\Program Files\HTML Help Workshop):

  • dans Afficher les dossiers pour choisissez Inclure les fichiers, cliquez sur Nouveau et tapez
    C:\Program Files\HTML Help Workshop\include

  • dans Afficher les dossiers pour choisissez Fichiers de librairies, cliquez sur Nouveau et tapez
    C:\Program Files\HTML Help Workshop\lib

  • dans Afficher les dossiers pour choisissez Fichiers exécutables, cliquez sur Nouveau et tapez
    C:\Program Files\HTML Help Workshop

  • fermez DevStudio et redémarrez pour enregistrer les changements.

Pour compiler le script d'installation, vous devrez télécharger et installer NSIS. Après l'installation de NSIS, faites un clic droit sur le fichier FileZilla.nsi dans l'explorateur et choisissez "Compiler NSI".
