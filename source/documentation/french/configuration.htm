Configuration

Vue d'Ensemble:

FileZilla supporte beaucoup d'options de configuration que vous pouvez régler dans la boîte de dialogues "Options de FileZilla".  Cliquez sur "Paramètres" dans le menu "Édition" pour ouvrir la boîte de dialogue.

Dans des conditions normales, vous n'avez pas à configurer FileZilla. FileZilla est fourni avec des réglages d'options par défaut qui sont optimisés pour la plupart des utilisateurs.

Les options sont divisées en plusieurs sections énumérées à gauche de la boîte de dialogue des Options. Cliquez sur une entrée de l'arborescence pour ouvrir la section correspondante.


(Fenêtre de connexion de la boîte de dialogue des Options de FileZilla)

Chaque page présente des rubriques séparées et leurs options associées:

  • Connexion - Cette page présente certains options générales de connexion
  • Configuration du Pare-feu - Configure les connexions à travers un pare-feu
  • Configuration Proxy FTP Proxy - Configure à travers un proxy FTP
  • Configuration Proxy - Configure à travers les proxy SOCKS4/5
  • Support GSS - Options pour le support GSS intégré
  • Cache de Répertoire - Active/désactive le cache local de la structure du serveur distant
  • Serveur d'Identification - Active/désactive le serveur d'identification intégré.
  • Configuration des Transferts de Fichiers - Contrôle les différents aspects du transfert des fichiers
  • Transfert ASCII/Binaire - Sélection du mode de transfert
  • Limite de Vitesse - Sélection du mode de transfert
  • Configuration de l'Interface - Contrôle l'affichage des fenêtres de FileZilla
  • Liste des Fichiers Locaux - Configure la disposition de la liste de fichiers locaux
  • Liste des Fichiers Distants - Configure la disposition de la liste de fichiers distants
  • Langue - Choisit la langue de FileZilla
  • Divers - Options supplémentaires concernant l'interface
  • Afficher /Éditer un Fichier - Installe les associations de type de fichier pour afficher / éditer les fichiers distants
