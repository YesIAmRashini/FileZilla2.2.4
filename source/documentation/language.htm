Language

Overview:

FileZilla currently supports two languages: English and German.  It uses English by default.  In order to display German, you must download the German language DLL from the FileZilla homepage page.  We welcome any of you to translate FileZilla and/or this documentation to your language of choice! :-)

In the Language page of the FileZilla Options dialog box, you can select the language you wish FileZilla to display in.

Simply click your language, and then click OK to close the dialog box.
